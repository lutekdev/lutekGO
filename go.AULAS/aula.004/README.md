> Para saber se uma função é pública ou privada é considerado pela primeira letra da função:

- Letra Minuscula:
    - Função Privada:
        - Só vai reconhecer dentro do arquivo onde está sendo criada.
    - Função Pública:
        - Pode ser exportada em qualquer arquivo.
